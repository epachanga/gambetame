How to play

The idea of the game is to guess as many game results as possible and compete with your friends to see how has better knowledge about football.

Game play
  1. Login with facebook
  2. Create a group or join a friends group*
  3. Start guessing matches results all the way to the final**
  4. Don't forget to save :)
  5. Compare with your group friends how scored better
* You can create/join more the one group.
** Choose Simple mode if you only want to guess who the winner will be without guessing the match score.

Scoring system
Advanced mode:
  1. If you guess the correct winner you get 2 points
  2. If you guess a tie (in group round) you get 2 points
  3. If you guess the exact match score you get 1 point
  4. If you guess the exact amount of goals for a team you get 1 point
  5. If you guess the amount of goals will be 4 or more you get 1 point
Simple mode:
  1. If you guessed the correct winner/tie you get 2 points
